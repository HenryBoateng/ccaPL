Context expressions are described by the following language:
κ ::= Context Expressions
0 empty context
true true
false false
n = m name match
this hole
not κ negation
κ1 | κ2 parallel composition
κ1 and κ2 conjunction
κ1 or κ2 disjunction
n[κ] location
next κ spatial next modality
somewhere κ somewhere modality
The inactivity process or null process 0 does nothing terminates immediately is the unit element for the parallel composition of processes, i.e. 0 | P ≡ P | 0 ≡ P where the symbol ‘≡’ means ‘is equivalent to’.
